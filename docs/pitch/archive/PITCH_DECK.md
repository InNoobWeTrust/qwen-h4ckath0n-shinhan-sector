# How It Works

1. Customer clicks **"Pay with Shinhan Installments"**
2. **Qwen 7B LLM analyzes** full transaction context in real time
3. Risk model evaluates:
   ✅ Purchase category & intent
   ✅ Merchant trust score
   ✅ Device fingerprint, location, timing patterns
   ✅ Basket composition & user behavior
4. Decision: Approve/Deny/Partial limit
5. Customer selects payment plan
6. Merchant gets paid immediately
7. Customer is automatically on-boarded to SOL
