## Presentation Guidelines

1. Always state when you are using an industry estimate
2. Prefer qualitative statements over unsourced percentages
3. Use framing: "This is what users consistently report" instead of hard numbers
4. Never cite a percentage number without explicitly naming the source
5. Bankers respect transparency about what is known vs what is estimated

> **Key principle:** It is always better to say "we don't have verified public data for this, but this is what every user tells us" than to present an unsourced number as fact. Bankers will trust you more for being honest about limits of public data.
